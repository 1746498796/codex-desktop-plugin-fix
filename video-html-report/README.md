# HTML 视频报告页

这是一个 9:16 竖屏 HTML 动画页面，用来录制类似“AI 视频报告”的短视频。

## 使用方法

1. 打开 `index.html`
2. 浏览器全屏或调整为手机竖屏比例
3. 用 OBS / 剪映录屏 / 系统录屏录制 78 秒
4. 导入剪映，加 AI 配音和字幕

## 建议

- 录屏分辨率：1080x1920
- 帧率：30fps
- 配音使用 README 或 `AI_VIDEO_STORYBOARD.md` 中的口播稿
- 如果有真实截图，可以在剪映里覆盖到第 6 段验证画面附近


## 横屏版

如果不想做 9:16 竖屏，使用：

```text
landscape.html
```

建议录制：

- 分辨率：1920x1080
- 比例：16:9
- 时长：84 秒
- 用途：B 站 / YouTube / 横屏网页报告

横屏版更适合展示真实网页截图和路径信息。
